# 法人向けチャットボット

TypeScriptとNext.jsで実装された、法人向けウェブサイトに埋め込み可能な日本語対応チャットボットです。

## 機能

- 簡単な埋め込みスクリプト
- レスポンシブデザイン
- カスタマイズ可能なテーマ
- 定型質問のサポート
- 日本語完全対応

## 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/cayman1028/ChatbotA.git
cd ChatbotA

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
